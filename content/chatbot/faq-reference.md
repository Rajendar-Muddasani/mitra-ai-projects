# Chatbot FAQ Reference

## Common Questions & Answers

### Which project should I choose for my final year?
For B.Tech/BCA/MCA final year: Document Q&A Assistant is the most complete kit (fully available). It's intermediate level, uses RAG + FastAPI + OpenAI — very strong for interviews. For AI/ML branch, any Major Project Kit works.

### Is this free?
All course content and project kits are openly accessible. No payment required to access the learning content.

### Can I submit this as my college project?
Yes. These are learning tools — you build the project yourself using our guidance, code scaffold, and templates. The work is yours. We provide the structure, not the finished product.

### Will this guarantee placement or marks?
No. We provide the tools, code structure, explanation system, and viva prep. Outcomes depend on effort and the student's college/company requirements.

### What is a viva?
A viva is an oral examination where you defend your project in front of faculty or examiners. You need to explain your architecture, code choices, and handle technical questions. Our viva Q&A guides help prepare for this.

### What if I'm a beginner?
Start with the Programming Foundation course (Python Basics tab), then the Machine Learning course. Once you understand the concepts, pick a Mini Project or Portfolio Build.

### How does the certificate work?
Pass all topic quizzes in a course (70%+) to earn a course completion certificate. Complete all courses to earn the master certificate. Certificates can be verified at mitraaiprojects.com/verify.html.

### How do notebooks work?
Click "Open Notebook" on any course topic. A loading page opens then launches JupyterLite (full Python in your browser — no installation needed). First load takes 30–60 seconds (Pyodide Python runtime downloads). After that it's fast. Your work saves automatically in the browser.

### What is Pyodide?
Pyodide is Python compiled to WebAssembly. It runs Python completely in your browser without any server or installation. The first load downloads ~30MB but this is cached for future visits.

### What courses cover deep learning?
The Deep Learning course covers Neural Networks, CNNs, RNNs/LSTMs, Autoencoders, GANs, Transformers, Multimodal Models, Transfer Learning, and Model Compression.

### What is RAG?
RAG (Retrieval-Augmented Generation) retrieves relevant documents from a knowledge base and injects them into an LLM prompt. This grounds the LLM in real information rather than its training data, reducing hallucination. The Document Q&A Assistant project is built on RAG.

### How do I run notebooks on GPU?
The JupyterLite notebooks run in your browser (CPU only). For GPU training, use Kaggle Notebooks (30 hours/week free T4 GPU) or Google Colab. The Key Concepts course covers GPU training on Kaggle.

### What is the difference between courses and projects?
Courses = concept-first learning (understand how and why). Projects = outcome-first builds (build something real). Both cross-link to each other. Use courses to understand the tech, use projects to apply it.
