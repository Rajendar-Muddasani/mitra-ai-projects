## Course Features
- OneNote-style left-sidebar layout (tabs on left, content on right)
- JupyterLite interactive notebooks embedded (opens in new tab)
- 10-question quiz per topic (pass 70% to mark complete)
- One course completion certificate per course
- One master certificate for completing all courses
